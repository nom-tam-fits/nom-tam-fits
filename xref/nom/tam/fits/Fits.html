64  /**
65   * This class provides access to routines to allow users to read and write FITS
66   * files. <br>
67   * <b> Description of the Package </b>
68   * <p>
69   * This FITS package attempts to make using FITS files easy, but does not do
70   * exhaustive error checking. Users should not assume that just because a FITS
71   * file can be read and written that it is necessarily legal FITS. These classes
72   * try to make it easy to transform between arrays of Java primitives and their
73   * FITS encodings.
74   * <ul>
75   * <li>The Fits class provides capabilities to read and write data at the HDU
76   * level, and to add and delete HDU's from the current Fits object. A large
77   * number of constructors are provided which allow users to associate the Fits
78   * object with some form of external data. This external data may be in a
79   * compressed format.
80   * <p>
81   * Note that this association is limited, it only specifies where the various
82   * read methods should read data from. It does not automatically read the data
83   * content and store the results. To ensure that the external content has been
84   * read and parsed the user may wish to invoke the read() method after creating
85   * the Fits object associated with external data. E.g.,
86   * 
87   * <pre>
88   *     File fl = ...  ; 
89   *     Fits f = new Fits(fl); // Or we could have used one of the other constructors.
90   *     // At this point the Fits object is empty.
91   *     f.read();    // Read the external data into the Fits object
92   *     // At this point the Fits object should have one or more HDUs depending
93   *     // upon the external content.
94   * </pre>
95   * 
96   * Users can choose to read only some of the HDUs in a given input, and may add
97   * HDU's that were either read from other files or generated by the program. See
98   * the various read and addHDU methods.
99   * <li>The FitsFactory class is a factory class which is used to create HDUs.
100  * HDU's can be of a number of types derived from the abstract class BasicHDU.
101  * The hierarchy of HDUs is:
102  * <ul>
103  * <li>BasicHDU
104  * <ul>
105  * <li>ImageHDU
106  * <li>RandomGroupsHDU
107  * <li>TableHDU
108  * <ul>
109  * <li>BinaryTableHDU
110  * <li>AsciiTableHDU
111  * </ul>
112  * <li>UndefinedHDU
113  * </ul>
114  * </ul>
115  * <li>The Header class provides many functions to add, delete and read header
116  * keywords in a variety of formats.
117  * <li>The HeaderCard class provides access to the structure of a FITS header
118  * card.
119  * <li>The header package defines sets of enumerations that allow users to
120  * create and access header keywords in a controlled way.
121  * <li>The Data class is an abstract class which provides the basic methods for
122  * reading and writing FITS data. It provides methods to get the the actual
123  * underlying arrays and detailed methods for manipulation specific to the
124  * different data types.
125  * <li>The TableHDU class provides a large number of methods to access and
126  * modify information in tables.
127  * <li>The utilities package includes simple tools to copy and list FITS files.
128  * </ul>
129  * 
130  * @version 1.16
131  */

251     /**
252      * Associate the FITS object with a file or URL. The string is assumed to be
253      * a URL if it begins one of the protocol strings. If the string ends in .gz
254      * it is assumed that the data is in a compressed format. All string
255      * comparisons are case insensitive.
256      * 
257      * @param filename
258      *            The name of the file or URL to be processed. The content of
259      *            this file will not be read into the Fits object until the user
260      *            makes some explicit request.
261      * @param compressed
262      *            is the file compressed?
263      * @throws FitsException
264      *             Thrown if unable to find or open a file or URL from the
265      *             string given.
266      **/
267     @SuppressWarnings("resource")
268     public Fits(String filename, boolean compressed) throws FitsException {
269         if (filename == null) {
270             throw new FitsException("Null FITS Identifier String");
271         }
272         try {
273             File fil = new File(filename);
274             if (fil.exists()) {
275                 fileInit(fil, compressed);
276                 return;
277             }
278         } catch (Exception e) {
279             LOG.log(Level.FINE, "not a file " + filename, e);
280             throw new FitsException("could not detect type of " + filename, e);
281         }
282         try {
283             InputStream str = Thread.currentThread().getContextClassLoader().getResourceAsStream(filename);
284             if (str != null) {
285                 streamInit(str);
286                 return;
287             }
288         } catch (Exception e) {
289             LOG.log(Level.FINE, "not a resource " + filename, e);
290             throw new FitsException("could not detect type of " + filename, e);
291         }
292         try {
293             InputStream is = FitsUtil.getURLStream(new URL(filename), 0);
294             streamInit(is);
295             return;
296         } catch (Exception e) {
297             LOG.log(Level.FINE, "not a url " + filename, e);
298             throw new FitsException("could not detect type of " + filename, e);
299         }
300 
301     }

429     /**
430      * Delete an HDU from the HDU list.
431      * 
432      * @param n
433      *            The index of the HDU to be deleted. If n is 0 and there is
434      *            more than one HDU present, then the next HDU will be converted
435      *            from an image to primary HDU if possible. If not a dummy
436      *            header HDU will then be inserted.
437      * @throws FitsException
438      *             if the HDU could not be deleted.
439      */
440     public void deleteHDU(int n) throws FitsException {
441         int size = getNumberOfHDUs();
442         if (n < 0 || n >= size) {
443             throw new FitsException("Attempt to delete non-existent HDU:" + n);
444         }
445         this.hduList.remove(n);
446         if (n == 0 && size > 1) {
447             BasicHDU<?> newFirst = this.hduList.get(0);
448             if (newFirst.canBePrimary()) {
449                 newFirst.setPrimaryHDU(true);
450             } else {
451                 insertHDU(BasicHDU.getDummyHDU(), 0);
452             }
453         }
454     }

480     /**
481      * Returns the n'th HDU. If the HDU is already read simply return a pointer
482      *         to the cached data. Otherwise read the associated stream until
483      *         the n'th HDU is read.
484      * @param n
485      *            The index of the HDU to be read. The primary HDU is index 0.
486      * @return The n'th HDU or null if it could not be found.
487      * @throws FitsException
488      *             if the header could not be read

491      *             
492      * @see #getHDU(String)
493      * @see #getHDU(String, int)
494      */

496         for (int i = getNumberOfHDUs(); i <= n; i++) {
497             BasicHDU<?> hdu = readHDU();
498             if (hdu == null) {
499                 return null;
500             }
501         }
502         return this.hduList.get(n);
503     }

505     /**
506      * Checks if the value of the EXTNAME keyword of the specified HDU matches the specified name.
507      * 
508      * @param hdu   The HDU whose EXTNAME to check
509      * @param name  The expected name
510      * @return      <code>true</code> if the HDU has an EXTNAME keyword whose value matches the
511      *              specified name (case sensitive!), otherwise <code>false</code>
512      *              
513      * @see #getHDU(String)
514      */
515     private boolean isNameMatch(BasicHDU<?> hdu, String name) {
516         Header h = hdu.getHeader();
517         if (!h.containsKey(EXTNAME)) {
518             return false;
519         }
520         return name.equals(hdu.getHeader().getStringValue(EXTNAME));
521     }
522     
523     /**
524      * Checks if the value of the EXTNAME and EXTVER keywords of the specified HDU match the specified name 
525      * and version.
526      * 
527      * @param hdu       The HDU whose EXTNAME to check
528      * @param name      The expected name
529      * @param version   The expected extension version
530      * @return          <code>true</code> if the HDU has an EXTNAME keyword whose value matches the
531      *                  specified name (case sensitive!) AND has an EXTVER keyword whose value
532      *                  matches the specified integer version. In all other cases <code>false</code>
533      *                  is returned.
534      *                  
535      * @see #getHDU(String, int)
536      */
537     private boolean isNameVersionMatch(BasicHDU<?> hdu, String name, int version) {
538         Header h = hdu.getHeader();
539         if (!h.containsKey(EXTNAME)) {
540             return false;
541         }
542         if (!name.equals(h.getStringValue(EXTNAME))) {
543             return false;
544         }
545         if (!h.containsKey(EXTVER)) {
546             return false;
547         }
548         return h.getIntValue(EXTVER) == version;
549     }

551     /**
552      * Returns the HDU by the given extension name (defined by EXTNAME). This method checks only for EXTNAME
553      * but will ignore the version (defined by EXTVER). If multiple HDUs have the same matching EXTNAME,
554      * this method will return the first match only.
555      * 
556      * @param name          The name of the HDU as defined by EXTNAME (case sensitive)
557      * @return              The first HDU that matches the specified extension name and version, or <code>null</code> if
558      *                      the FITS does not contain a matching HDU.
559      * @throws FitsException    if the header could not be read
560      * @throws IOException      if the underlying buffer threw an error
561      * 
562      * @since 1.17.0
563      * 
564      * @see #getHDU(String, int)
565      * @see #getHDU(int)
566      */
567     public BasicHDU<?> getHDU(String name) throws FitsException, IOException {
568         // Check HDUs we already read...
569         for (BasicHDU<?> hdu : hduList) {
570             if (isNameMatch(hdu, name)) {
571                 return hdu;
572             }
573         }
574         
575         // Read additional HDUs as necessary...
576         BasicHDU<?> hdu;
577         while ((hdu = readHDU()) != null) {
578             if (isNameMatch(hdu, name)) {
579                 return hdu;
580             }
581         }
582         
583         return null;
584     }
585     
586     /**
587      * Returns the HDU by the given extension name and version (defined by EXTNAME and EXTVER keywords).
588      * If multiple HDUs have the same matching name and version, this method will return the first match only.
589      * 
590      * @param name          The name of the HDU as defined by EXTNAME (case sensitive)
591      * @param version       The extension version as defined by EXTVER in the matching HDU.
592      * @return              The first HDU that matches the specified extension name and version, or <code>null</code> if
593      *                      the FITS does not contain a matching HDU.
594      * @throws FitsException    if the header could not be read
595      * @throws IOException      if the underlying buffer threw an error
596      * 
597      * @since 1.17.0
598      * 
599      * @see #getHDU(String)
600      * @see #getHDU(int)
601      */
602     public BasicHDU<?> getHDU(String name, int version) throws FitsException, IOException {
603         // Check HDUs we already read...
604         for (BasicHDU<?> hdu : hduList) {
605             if (isNameVersionMatch(hdu, name, version)) {
606                 return hdu;
607             }
608         }
609         
610         // Read additional HDUs as necessary...
611         BasicHDU<?> hdu;
612         while ((hdu = readHDU()) != null) {
613             if (isNameVersionMatch(hdu, name, version)) {
614                 return hdu;
615             }
616         }
617         
618         return null;
619     }

641     /**
642      * Insert a FITS object into the list of HDUs.
643      * 
644      * @param myHDU
645      *            The HDU to be inserted into the list of HDUs.
646      * @param position
647      *            The location at which the HDU is to be inserted.
648      * @throws FitsException
649      *             if the HDU could not be inserted.
650      */
651     public void insertHDU(BasicHDU<?> myHDU, int position) throws FitsException {
652         if (myHDU == null) {
653             return;
654         }
655         if (position < 0 || position > getNumberOfHDUs()) {
656             throw new FitsException("Attempt to insert HDU at invalid location: " + position);
657         }
658         try {
659             if (position == 0) {
660                 // Note that the previous initial HDU is no longer the first.
661                 // If we were to insert tables backwards from last to first,
662                 // we could get a lot of extraneous DummyHDUs but we currently
663                 // do not worry about that.
664                 if (getNumberOfHDUs() > 0) {
665                     this.hduList.get(0).setPrimaryHDU(false);
666                 }
667                 if (myHDU.canBePrimary()) {
668                     myHDU.setPrimaryHDU(true);
669                     this.hduList.add(0, myHDU);
670                 } else {
671                     insertHDU(BasicHDU.getDummyHDU(), 0);
672                     myHDU.setPrimaryHDU(false);
673                     this.hduList.add(1, myHDU);
674                 }
675             } else {
676                 myHDU.setPrimaryHDU(false);
677                 this.hduList.add(position, myHDU);
678             }
679         } catch (NoSuchElementException e) {
680             throw new FitsException("hduList inconsistency in insertHDU", e);
681         }
682     }

745     /**
746      * Read the next HDU on the default input stream.
747      * 
748      * @return The HDU read, or null if an EOF was detected. Note that null is
749      *         only returned when the EOF is detected immediately at the
750      *         beginning of reading the HDU.
751      * @throws FitsException
752      *             if the header could not be read
753      * @throws IOException
754      *             if the underlying buffer threw an error
755      */
756     public BasicHDU<?> readHDU() throws FitsException, IOException {
757         if (this.dataStr == null || this.atEOF) {
758             if (this.dataStr == null) {
759                 LOG.warning("trying to read a hdu, without an input source!");
760             }
761             return null;
762         }
763         if (this.dataStr instanceof RandomAccess && this.lastFileOffset > 0) {
764             FitsUtil.reposition(this.dataStr, this.lastFileOffset);
765         }
766 
767         Header hdr = Header.readHeader(this.dataStr);
768         if (hdr == null) {
769             this.atEOF = true;
770             return null;
771         }
772 
773         Data data = hdr.makeData();
774         try {
775             data.read(this.dataStr);
776         } catch (PaddingException e) {
777             // Stream end before required padding after data...
778             e.updateHeader(hdr);
779             LOG.warning("Missing padding after data segment.");
780         }
781 
782 
783         // Check for truncation even if we successfully skipped to the expected end
784         // since skip may allow going beyond the EOF.
785         if (Fits.checkTruncated(this.dataStr)) {
786             // File ends before required padding after data...
787             // (files allow skipping beyond the end, which is why we don't
788             // catch it above)
789             LOG.warning("Missing padding after data segment.");
790         }
791 
792         this.lastFileOffset = FitsUtil.findOffset(this.dataStr);
793         BasicHDU<Data> nextHDU = FitsFactory.hduFactory(hdr, data);
794         this.hduList.add(nextHDU);
795 
796         return nextHDU;
797     }

799     /**
800      * Read to the end of the associated input stream
801      * 
802      * @throws FitsException
803      *             if the operation failed
804      */
805     private void readToEnd() throws FitsException {
806         try {
807             while (this.dataStr != null && !this.atEOF) {
808                 if (readHDU() == null) {
809                     if (getNumberOfHDUs() == 0) {
810                         throw new FitsException("Not FITS file.");
811                     }
812                     return;
813                 }
814             }
815         } catch (IOException e) {
816             throw new FitsException("Corrupted FITS file.", e);
817         }

820     /**
821      * <p>
822      * Add or modify the CHECKSUM keyword in all headers. As of 1.17 the checksum for deferred
823      * data is calculated directly from the file (if possible), without loading the entire 
824      * (potentially huge) data into RAM for the calculation.
825      * </p>
826      * <p>
827      * As of 1.17, the routine calculates checksums both for HDUs that are in RAM, as well as
828      * HDUs that were not yet loaded from the input (if any). Any HDUs not in RAM at the
829      * time of the call will stay in deferred mode (if the HDU itself supports it). After
830      * setting (new) checksums, you may want to call #rewrite()
831      * </p>
832      * 
833      * @throws FitsException
834      *             if the operation failed
835      * @throws IOException
836      *             if the underlying stream failed
837      *             
838      * @author R J Mather, Attila Kovacs
839      * 
840      * @see #rewrite()
841      */
842     public void setChecksum() throws FitsException, IOException {
843         int i = 0;
844         
845         // Start with HDU's already loaded, leaving deferred data in unloaded state
846         for (; i < getNumberOfHDUs(); i++) {
847             BasicHDU<?> hdu = getHDU(i);
848             FitsCheckSum.setDatasum(hdu.getHeader(), calcDatasum(i));
849         }
850         
851         // Check if Fits is read from an input of sorts, with potentially more HDUs there...
852         if (dataStr == null) {
853             return;
854         }
855         
856         // Continue with unread HDUs (if any...)
857         BasicHDU<?> hdu = null;
858         while ((hdu = readHDU()) != null) {
859             FitsCheckSum.setDatasum(hdu.getHeader(), calcDatasum(i++));
860         }
861     }

863     /**
864      * Calculates the data checksum for a given HDU in the Fits. If the HDU does not currently
865      * have data loaded from disk (in deferred read mode), the method will calculate the
866      * checksum directly from disk. Otherwise, it will calculate the datasim from the data
867      * in memory.
868      * 
869      * @param hduIndex          The index of the HDU for which to calculate the data checksum
870      * @return                  The data checksum, e.g. for comparing against
871      *                          {@link BasicHDU#getStoredDatasum()}
872      * @throws FitsException    if there was an error processing the HDU.
873      * @throws IOException      if there was an I/O error accessing the input.
874      * 
875      * @see #calcChecksum(int)
876      * @see BasicHDU#getStoredDatasum()
877      * @see #setChecksum()
878      * @see FitsCheckSum#checksum(Data)
879      * @see FitsCheckSum#setDatasum(Header, long)
880      * 
881      * @since 1.17
882      * 
883      */
884     public long calcDatasum(int hduIndex) throws FitsException, IOException {
885         BasicHDU<?> hdu = getHDU(hduIndex);
886         Data data = hdu.getData();
887         if (data.isDeferred()) {
888             // Compute datasum directly from file...
889             return FitsCheckSum.checksum((RandomAccess) dataStr, data.getFileOffset(), data.getSize());
890         } 
891         return FitsCheckSum.checksum(data);
892     }

894     /**
895      * Calculates the FITS checksum for a given HDU in the Fits. If the HDU does not currently
896      * have data loaded from disk (i.e. in deferred read mode), the method will compute the
897      * checksum directly from disk. Otherwise, it will calculate the checksum from the data
898      * in memory.
899      * 
900      * @param hduIndex          The index of the HDU for which to calculate the HDU checksum
901      * @return                  The HDU's checksum, e.g. for comparing against
902      *                          {@link BasicHDU#getStoredChecksum()}
903      * @throws FitsException    if there was an error processing the HDU.
904      * @throws IOException      if there was an I/O error accessing the input.
905      * 
906      * @see #calcDatasum(int)
907      * @see BasicHDU#getStoredChecksum()
908      * @see #setChecksum()
909      * @see FitsCheckSum#setChecksum(BasicHDU)
910      * 
911      * @since 1.17
912      * 
913      */
914     public long calcChecksum(int hduIndex) throws FitsException, IOException {
915         return FitsCheckSum.sumOf(FitsCheckSum.checksum(getHDU(hduIndex).getHeader()), calcDatasum(hduIndex));
916     }

1229     /**
1230      * Checks if the file ends before the current read positon, and if so, it 
1231      * may log a warning. This may happen with {@link FitsFile} where the 
1232      * contract of {@link RandomAccess} allows for skipping ahead beyond the 
1233      * end of file, since expanding the file is allowed when writing. Only a 
1234      * subsequent read call would fail.
1235      *
1236      * @param in    the input from which the FITS content was read.
1237      * @return      <code>true</code> if the current read position is beyond
1238      *              the end-of-file, otherwise <code>false</code>.
1239      * @throws IOException  if there was an IO error accessing the file or stream.
1240      * 
1241      * @see #skip(long)
1242      * @see #skipBytes(int)
1243      * @see #skipAllBytes(long)
1244      * 
1245      * 
1246      * @since 1.16
1247      */
1248     static boolean checkTruncated(ArrayDataInput in) throws IOException {
1249         if (!(in instanceof RandomAccess)) {
1250             // We cannot skip more than is available in an input stream.
1251             return false;
1252         }
1253         
1254         RandomAccess f = (RandomAccess) in;
1255         long pos = f.getFilePointer();
1256         long len = f.length();
1257         if (pos > len) {
1258             LOG.log(Level.WARNING, "Premature file end at " + len + " (expected " + pos + ")", new Throwable());
1259             return true;
1260         }
1261         return false;
1262     }
